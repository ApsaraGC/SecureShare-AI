# SecureShare AI

A simple cybersecurity practice project built using Python and Streamlit.

## Overview

SecureShare AI is a prototype application designed to demonstrate secure document handling concepts. The system allows users to upload documents, detect sensitive information, assess risk levels, and encrypt files before sharing.

This project was developed as part of my learning journey in cybersecurity and preparation for the Cyber Defence Innovation Hackathon.

## Features

* Document Upload
* Sensitive Data Detection
* Email Detection
* Phone Number Detection
* Risk Assessment
* File Encryption
* Secure Document Handling

## Technologies Used

* Python
* Streamlit
* Cryptography
* Regular Expressions (Regex)

## Installation

Install the required packages:

```bash
pip install streamlit cryptography pandas
```

## Running the Application

```bash
streamlit run app.py
```

The application will be available at:

```text
http://localhost:8501
```
